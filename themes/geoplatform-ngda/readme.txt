== Description ==

This is the theme for the Geoplatform Community Core Bundle.

== Installation ==

1. In your admin panel, go to Appearance > Themes and the theme should already be bundled in your package. If it is not, contact the Geoplatform Tech Team about your installation.
2. Navigate to "GeoPlatform 2017 Theme".
3. Click Activate to use your theme right away.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

GeoPlatform Test includes support for Infinite Scroll in Jetpack.
